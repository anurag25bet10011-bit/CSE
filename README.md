# Income and Expense Tracking Program

A simple Python program to track your *income and expenses, calculate your **net balance, and visualize your finances using **bar* and *pie charts*.

This is a console-based tool where you manually enter your transactions (income or expense) and get a quick summary of your financial situation.

##  Features

- Add multiple *income* and *expense* entries
- Store:
  - Type of transaction (income / expense)
  - Amount
  - Category (e.g., Food, Rent, Salary)
- Automatically calculates:
  - Total Income
  - Total Expense
  - Net Balance
  - Simple financial status message:
    - “You have saved money.” or  
    - “Your spending is greater than your earnings.”
- Visualizations using Matplotlib:
  - *Bar chart* comparing total income vs total expense
  - *Pie chart* showing category-wise expense distribution


##  Technologies Used

- *Python*
- *NumPy* – for numeric operations and array handling
- *Matplotlib* – for plotting graphs
